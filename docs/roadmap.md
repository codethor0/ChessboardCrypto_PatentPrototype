# Roadmap

## Current Release

- Vendor-neutral public README and architecture docs
- State-tree validation POC with JSON examples
- Chessboard embodiment validation pipeline
- Docker lab and GitHub Actions validation workflow
- Public staging bundle script with hygiene checks

## Near Term

- Additional example graphs for SOAR and agentic workflow gating
- Expanded path transcript and audit record examples
- Optional GitHub Pages site from `docs/index.md`
- HTML export of public docs if useful for offline review

## Medium Term

- Threshold secret-sharing example for key-share binding
- Graph mutation examples tied to policy version changes
- Separate public repository name aligned with final GitHub org decision
- Independent cryptographic review checklist for external reviewers

## Explicit Non-Goals

- Production deployment into live security products
- Vendor-specific integrations
- Claims of NIST validation or cryptographic proof
- Automatic inclusion of private patent committee PDFs or DOCX drafts in public branches

## Legal and Disclosure Track

Parallel to engineering work:

- formal prior art search
- patent counsel review
- inventorship and assignment review
- public disclosure timeline confirmation

These items require qualified legal counsel and are not completed by this repository alone.
